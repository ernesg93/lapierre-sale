# Specification: Technical Sections

**Change**: technical-sections-tests

## Requirements

### TechSpecs
- MUST display the title "Ficha Técnica".
- MUST render all specifications defined in the component (Cuadro, Grupo, Ruedas, Frenos, Talla, Peso, Año, Uso, Kilometraje, Estado).
- Labels MUST be visible and values MUST match the provided data.

### TrustSection
- MUST display the title "Compra con Confianza".
- MUST render the 4 trust badges (Factura, Nº de serie, Prueba, Pago).
- MUST show the "Verificado para venta local" badge with its pulsing animation container.
